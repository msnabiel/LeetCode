# LeetCode Solutions

Welcome to my **LeetCode Solutions Repository**! This repository contains solutions to problems I've solved on [LeetCode](https://leetcode.com/). It serves as a personal library for tracking progress, learning, and sharing my approaches to various coding challenges.

## 📂 Repository Structure

The repository is organized into the following structure:

```
LeetCode/
├── Easy/
├── Medium/
├── Hard/
├── Topics/
│   ├── Arrays/
│   ├── Strings/
│   ├── Dynamic_Programming/
│   ├── Backtracking/
│   ├── …
├── Utilities/
│   ├── readme_generator.py
│   ├── leetcode_template.py
└── README.md
```

### **Folders:**

- **Easy**, **Medium**, and **Hard**:
  - Solutions are categorized by difficulty level
  - Each folder contains problem-specific Python scripts named `<problem-name>.py`
- **Topics**:
  - Solutions are grouped by topics such as `Arrays`, `Strings`, `Graphs`, and `Dynamic Programming`
  - This helps in quickly revisiting problems based on concepts
- **Utilities**:
  - Contains utility scripts:
    - `readme_generator.py`: Script for auto-generating markdown files
    - `leetcode_template.py`: Template for writing LeetCode solutions

## 🚀 How to Use

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/msnabiel/LeetCode.git
   cd LeetCode
   ```

2. **Browse by Difficulty**:
   - Navigate to the Easy, Medium, or Hard folder to explore problems by difficulty level

3. **Browse by Topic**:
   - Visit the Topics folder to find problems categorized by data structures or algorithms

4. **Run a Solution**:
   - Use Python to run any solution file. For example:
   ```bash
   python Easy/two_sum.py
   ```
